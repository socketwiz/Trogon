Overview

Trogon is simply a wrapper around the RVM utility created by the wonderful folks at beginrescueend.com. If you're like me you have a hard time remembering all the various command line parameters. This utility aims to solve that problem by giving you the power of RVM at the click of a button. My primary motivation for this tool was for creating .rvmrc files. I would always have to lookup the syntax in a google search. Well no more, now I simply choose which directory I want it created in from a file chooser.
